# 悬浮导航

悬浮导航:一般悬浮于页面的右侧,功能有导航到粉丝页,下载,页面内活动导航等

配置项

```typescript
	_downloadBoxOption: {
    // 显示隐藏的距离
        distance: string;
        // 动画时间
        duration: number;
        // 位置
        direction: "right" | "left";
        // 是否有到顶部按钮
        isShowTop?: boolean;
        // 按钮
				btns: {
          // 页面内需要导航的互动
          // 活动id: "";
          // 打开新页面的下载和粉丝页
          // facebook: "https://www.facebook.com/PokeSagago/"
					[key: string]: string;
        };
        // 页面内需要导航的活动的id
        scrollElementIds:string[]
			};

```
